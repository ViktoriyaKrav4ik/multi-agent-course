# Корпус для RAG (агро / ринковий аналітик)

Покладіть сюди **10–20** документів: ваші **PDF** (звіти, презентації, аналітика), плюс за потреби `.md` / `.txt` (статті, витяги з Wikipedia).

- Рекомендовано підкаталог `corpus/reports/` для PDF, щоб не змішувати з службовими `.md`.
- Після додавання або зміни файлів: **`python ingest.py`** (з кореня `market-analyst/`).
- Якщо шлях до проєкту з кирилицею ламає FAISS на Windows — задайте в `.env` **`RAG_INDEX_DIR`** на ASCII-шлях (див. головний README).

У репозиторії залишено кілька коротких **seed**-файлів `agro_*.md`, щоб можна було швидко перевірити ingest без PDF; для здачі курсу замініть/доповніть їх своїми матеріалами.
