# Скріншоти Langfuse / LangSmith

Додайте сюди експорт з дашборду після прогонів:

- Trace повного пайплайну (`main.py`) з вкладеними викликами **analyst**, **critic**, **compiler**.
- Приклад циклу **Analyst ↔ Critic** (кілька ітерацій): видно `metadata.analyst_critic_iteration` та `session_id`.

**Демо:** запишіть відео або GIF повного запуску (згідно з ТЗ курсу) та покладіть у репозиторій або надішліть лінк у коментарі до здачі.
